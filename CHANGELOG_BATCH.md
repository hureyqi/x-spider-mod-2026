### 🔧 技术实现

#### 数据结构
```typescript
interface BatchList {
  id: string;
  name: string;
  description?: string;
  accounts: string[];
  filter: {
    mediaTypes: ('photo' | 'video' | 'gif')[];
    source: 'medias' | 'tweets';
  };
  createdAt: number;
  updatedAt: number;
  lastUsedAt?: number;
}
```

#### 核心功能
1. **列表管理**：完整的 CRUD 操作，数据持久化
2. **账户导入**：支持多种格式（换行、逗号、分号）
3. **批量下载**：异步串行下载，错误处理和重试
4. **进度追踪**：实时更新，支持暂停/继续
5. **文件夹分类**：自动按博主用户名创建文件夹

### 📱 使用流程

1. **创建列表**
   - 点击"创建新列表"或"快速创建"
   - 填写列表名称和描述
   - 选择媒体类型（图片/视频/GIF）
   - 选择下载源（媒体/帖子）

2. **导入账户**
   - 粘贴账户名列表
   - 支持拖拽导入文件
   - 实时预览和验证
   - 一键导入

3. **开始下载**
   - 选择列表，点击"下载"
   - 系统自动按顺序下载每个账户
   - 自动创建文件夹分类
   - 实时显示进度和日志

4. **监控进度**
   - 查看总进度条
   - 查看成功/失败统计
   - 查看详细日志
   - 随时暂停/继续/停止
