# 🎯 X-Spider 批量下载功能使用指南

## 快速开始

### 第一步：创建批量列表

1. 在侧边栏点击 **"批量下载"**
2. 点击 **"创建新列表"** 或 **"快速创建"**
3. 填写列表信息：
   - 列表名称：如"我的收藏"
   - 描述：可选
   - 媒体类型：选择图片/视频/GIF
   - 下载源：选择"媒体"或"帖子"

### 第二步：添加账户

**方法1：手动添加**
- 在编辑列表时，点击输入框添加账户名
- 支持自动补全

**方法2：批量导入**
- 点击 **"导入账户"** 按钮
- 粘贴账户名列表（支持多种格式）
- 系统自动去重和验证

**方法3：文件导入**
- 拖拽 TXT 或 CSV 文件到上传区域
- 每行一个账户名

### 第三步：开始批量下载

1. 在列表中找到要下载的列表
2. 点击 **"下载"** 按钮
3. 系统自动：
   - 按顺序下载每个账户的媒体
   - 自动创建文件夹分类
   - 实时显示进度和日志

## 📊 功能特性

### 智能分类保存
- ✅ 自动按博主用户名创建文件夹
- ✅ 文件命名模板可自定义
- ✅ 自动跳过已下载文件

### 进度可视化
- 📈 **总体进度条**：显示所有账户的完成度
- 📊 **统计面板**：总数、已完成、失败、剩余
- ⏱️ **时间估算**：预计剩余时间
- 📝 **详细日志**：实时显示每个操作

### 灵活的下载控制
- ▶️ **开始下载**：从第一个账户开始
- ⏸️ **暂停下载**：暂停当前任务
- ▶️ **继续下载**：从暂停处继续
- ⏹️ **停止下载**：终止所有任务

### 错误处理
- ❌ 自动记录失败账户
- 🔄 可手动重试失败账户
- 📧 支持系统通知

## 🔧 故障排除

### 问题：下载失败

**解决方案**：
1. 检查网络连接
2. 确认代理设置正确
3. 检查账户名是否正确
4. 查看详细日志了解原因

### 问题：下载速度慢

**优化建议**：
1. 启用代理
2. 使用系统代理
3. 避开高峰期
4. 减少同时下载的账户数

### 问题：列表数据丢失

**原因**：未正确保存

**解决方案**：
1. 每次编辑后确保点击"保存"
2. 重启应用后检查数据是否保留
3. 定期备份重要列表

## 📝 最佳实践

### ✅ 推荐做法

1. **定期整理列表**
   - 删除不需要的账户
   - 合并相似的列表
   - 清理失败的记录

2. **使用描述信息**
   - 给列表添加清晰的描述
   - 方便以后快速识别

3. **分批下载**
   - 不要一次添加太多账户
   - 建议每批 20-30 个

4. **监控进度**
   - 定期查看下载日志
   - 及时处理失败任务

### ❌ 避免事项

1. 不要同时运行多个批量下载
2. 不要在下载过程中关闭应用
3. 不要添加无效的账户名
4. 不要忽视系统通知
